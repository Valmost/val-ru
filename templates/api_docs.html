REST API Документация

API для программного доступа к обработке PDF

Авторизация

Все защищенные эндпоинты требуют заголовок:

Authorization: Bearer ваш_токен
GET /api/status

Проверка работоспособности API

POST /api/token

Получение токена

{
    "username": "ваш_логин",
    "password": "ваш_пароль"
}
POST /api/process 🔒 Требуется токен

Обработка PDF (multipart/form-data)

curl -X POST http://localhost:5000/api/process \
  -H "Authorization: Bearer ваш_токен" \
  -F "file=@документ.pdf" \
  -F "width=100" \
  -F "height=200" \
  -F "algorithm=genetic" \
  -o результат.pdf
